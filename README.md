# Machine Learning Course Projects

This repository contains two machine learning projects completed as part of the course requirements.

---

## 1. XGBoost Classification – Bank Marketing Dataset

This project applies the XGBoost algorithm to classify customer responses in the Bank Marketing dataset.

**Dataset:**  
UCI Bank Marketing Dataset  
https://archive.ics.uci.edu/ml/datasets/Bank+Marketing

**Description:**
The project includes data preprocessing, feature encoding, model training, evaluation using accuracy and confusion matrix, and feature importance visualization.

---

## 2. Ensemble Learning – Customer Churn Prediction

This project implements ensemble learning techniques to predict customer churn using a supervised classification approach.

**Description:**
The notebook covers data preprocessing, model building with multiple classifiers, ensemble methods, and performance evaluation.
